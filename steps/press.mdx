title: Press

icon: command
---

For all available key combinations that can be used, see the [Key values](https://developer.mozilla.org/en-US/docs/Web/API/UI_Events/Keyboard_event_key_values) guide.

## Inputs

<ResponseField name="value" type="string" required>
  Keys that will be pressed using the keyboard such as `ArrowLeft` or
  `Control+Shift+T`. Multiple keys can be pressed at the same time by separating
  them with a `+`.
</ResponseField>
