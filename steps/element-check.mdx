Here are the types of checks you can perform on an element:

1. **Content** - Verify the text content of an element.
2. **Attribute** - Verify the value of an attribute on an element.
3. **State** - Verify the state of an element, such as whether it is visible or enabled.

## Inputs

<ResponseField name="description" type="string" required>
  Description of the element you want to assert on. Refer to the [Locating
  elements](/platform/locating-elements) guide for more information.
</ResponseField>
